Instance Method Details

- (Object) put(host, data, content_type = 'application/json') Also known as: submit

Submit data to Nagira API. Sets @response variable to [RestClient.response]. can submit single check or multiple checks for single host. If data is Array, then all the checks are submitted with one HTTP request.

Parameters:

  • host (String) —

    Hostname of the server check is executed against (not API Nagira host).

  • content_type (String) (defaults to: 'application/json') —

    HTTP Content-type string

  • data (Hash, Array) —

    Input data for the status check submission. Should contain attributes: service_description :return_code, :plugin_output. Note, attribute host_name is overriden by host parameter. Data acceptable by +Nagira#put_status_host_name_services+ method

Returns:

  • self

See Also:

  • Nagira#put_status_host_name_services

def put host, data, content_type='application/json'

  data = [data] if data.is_a? Hash # Make sure it's an Array

  data.each { |item|  item.merge!(:host_name => host) }
  @response = RestClient.put endpoint(host), data.to_json
  self
end
